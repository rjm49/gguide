Top 10 irregular verbs to know

In English, we use irregular verbs about 70% of the time. There are a lot of them, but you can learn them in order of importance.

Start with the Top Ten most common irregular verbs!

  • be -> was
  • have -> had
  • do -> did
  • say -> said
  • go -> went

  • get -> got
  • make -> made
  • know -> knew
  • think - thought
  • take -> took
